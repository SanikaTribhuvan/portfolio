🏆 FIRST PRIZE + BEST PAPER SCOPUS-INDEXED 🇮🇳 PATENT FILED SOLO

Phantom Ledgers: Detecting Hidden Manipulation in Public Budget Datasets via Cryptographic Integrity, Temporal Edit Graph Analysis & Agentic Alerting

ICIISTM 2026 — International Conference on Inventions, Innovations in Science, Technology & Management · March 2026

↑ Live system UI — phantom-ledger-2pjd.onrender.com

A 10-stage forensic AI pipeline for detecting hidden manipulation in Indian government budget data. Combines SHA-256 cryptographic hashing, Merkle trees, Bitcoin blockchain anchoring via OpenTimestamps, Temporal Edit Graph analysis, Isolation Forest ML triage, SHAP-style XAI with 5 explainability factors summing to 100%, union-find campaign detection, WhatsApp alerting under 200ms, and forensic PDF report generation. Validated on real Indian datasets — RBI, OpenBudgetsIndia, data.gov.in.

Indian Provisional Patent Filed · Sole Proprietor · ₹2,000 Cash Prize

🔗 Live System GitHub
🥈 SECOND PRIZE POSTER UGC-LISTED · IF 8.141 PUBLISHED · IJAAR SOLO

GlitchZero: AI-Native Middleware for Real-Time Institutional Data Integrity via Hybrid Anomaly Detection, Blockchain Anchoring & XAI

Published in IJAAR (International Journal of Advanced Academic Research) · Impact Factor 8.141 · NCRT-AIML 2026, SPPU · March 2026

↑ Live system UI — glitchzero.onrender.com

AI-native middleware system for real-time institutional database integrity detection. Uses hybrid Isolation Forest anomaly detection, SHAP-style Explainable AI, Merkle Tree blockchain anchoring on Polygon, and an Agentic Mitigation Engine. Metrics: F1 0.93, Precision 0.94, Recall 0.92, average latency 3.4ms.
